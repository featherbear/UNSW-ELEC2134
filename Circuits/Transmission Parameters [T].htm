Transmission parameters describe the transmission through the line, and use the parameters A, B, C and D

 

 

 

 

 

A = open-circuit reverse voltage gain;

B = negative short-circuit transfer impedance()

C = Open-circuit transfer admittance (S)

D = Negative short-circuit reverse current gain
